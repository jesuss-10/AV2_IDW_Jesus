HTML, CSS e JavaScript

Descubra o poder das tecnologias web!

HTML

O HTML é a estrutura do site. Ele define títulos, parágrafos, imagens, links e organiza todo o conteúdo.

<header>
  <h1>Título</h1>
  <p>Parágrafo</p>
</header>

CSS

O CSS é responsável pelo visual. Ele define cores, fontes, espaçamentos e deixa o site bonito e agradável.

.header {
  background: #3a86ff;
  color: white;
  padding: 20px;
}

JavaScript

O JavaScript traz interatividade. Com ele, é possível criar animações, validar formulários e muito mais!

function showMessage() {
  alert('Olá, mundo!');
}
